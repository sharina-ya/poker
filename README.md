
**Кнопки действий:**

FOLD - сбросить карты (выйти из раздачи)
CHECK - пропустить ход (если нет ставки для кола)
CALL - поднять до текущей ставки
RAISE - повысить ставку (введите сумму)
ALL-IN - поставить все фишки

**Фазы игры:**

Префлоп - получили по 2 карты, первый раунд торгов
Флоп - открываются 3 общие карты, второй раунд торгов
Тёрн - 4-я общая карта, третий раунд торгов
Ривер - 5-я общая карта, последний раунд торгов
Шоудаун - все оставшиеся игроки показывают карты
